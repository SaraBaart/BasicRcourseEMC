3.1 Selecting Data

Answer the following without typing the commands into R. Use ? if you’re not sure what the object is or what the function does

Question 1: length of objects

a

What is

length(letters)


b

What is

length(letters == LETTERS)

Question 2: which

a

What is

which(letters %in% c("a", "d"))

b

What is

which(c("a", 7, "d") %in% letters)

x <- matrix(8:11, nrow = 6, ncol = 4)

What are

x[, 3] + 2 * x[, 2]
nrow(x)

Question 7

Use R to answer the following:

Create a vector (using c()) called a (i.e. assign it to an object called a) with four elements which are the integers 5 to 8 (inclusive).

  1. Display element 2 of a.
  2. Display element 4 of a.
  3. Assign the integers 3 and 4 to object b and use b to select elements 3 and 4 of object a.
  4. Display every element of a except element 2.
  5. Display every element of a except elements 3 and 4.
  6. Display only those elements of a that are greater than or equal to 6.
  7. Display only those elements of a that are not equal to 7.
  8. Use the list function to create an object ab which is a list of the two objects a and b.
  9. Display ab.
  10. Change the names of the elements in ab to a and b.
  11. Display ab again. What has changed?

Question 8

  1. Create this matrix m:
m <- matrix(1:9, nrow = 3, byrow = T)
m
     [,1] [,2] [,3]
[1,]    1    2    3
[2,]    4    5    6
[3,]    7    8    9
  1. Why are the numbers 1, 2, and 3 in the first row and not the first column?
  2. Display the element on the second row and second column of m.
  3. Display only the 2nd row of m.
  4. Display only the 3rd column of m.
  5. Display only the 2nd and 3rd columns of m. Do so in two different ways.

Question 9

Using the heart data set (remember to load the survival package):

  • Select the first row.
  • Select the first column.
  • Select the column surgery.

Question 10

Create a list:

x <- c(1,2,3,4)
mat <- matrix(1:9, 3, 3)
myList <- list(a=x, b=mat)

  1. What is the class of myList[[2]]

  2. What is the class of myList[2]

  3. Write R code to extract the first element (the vector) by name

If you want to save your work: save your R session and/or source code!
